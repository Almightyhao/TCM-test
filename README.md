# TCM Herb Quiz

這是一個互動式中藥飲片測驗網頁，收錄 **161 種院內常見飲片**，搭配圖片與分類，幫助使用者透過問答遊戲熟悉藥材名稱與功效分類。  

## 功能
- 題目隨機出題，可自訂題數
- 顯示飲片圖片與四選一選項
- 答題即時回饋，完成後顯示分數與評語
- 圖片載入失敗時會提示錯誤

## 使用方式
1. 開啟 `index.html`
2. 輸入本次想要測驗的題數
3. 開始答題並查看分數結果
